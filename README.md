# self-vue-app-base [WIP]

My base application with vue.js and

- haml
- coffee
- sass
- grunt(for tool)

## Setup

```console
git clone
$ gem install haml
$ npm install -g yo grunt-cli bower
$ npm install
$ bower install
```

## For development

```console
$ grunt serve
```

## build static files

```console
$ grunt build
```

Static files are created at dist/ directory.
